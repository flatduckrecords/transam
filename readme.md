# ZXtranSAM
A ZX Spectrum to SAM Coupé BASIC translator and variable importer.

[About ZXtranSAM](https://robertmorrison.me/sam/zxtransam/)

Based on:
- BTRANS by Dr Andrew Wright (1989)
- Variables List by Neil R. Canham (1984)
